Listas Ordenadas

Para criarmos lista ordenada (orderd list), usamos a tag <ol> para delimitar a lista e <li> (list item) para indentificar cada item da lista.

A tag <ol> possui um parâmetro type (tipo), onde configuramos o tipo de marcador da lista atual. Podemos indicar o ínicio da contagem usando o parâmetro start.

Listas numeradas.

Para criar listas numeradas ultizamos o TYPE 1.

        
<ol type = "1" start = "6"> <!--a tag start indica em qual contagem a lista vai começar.-->
    <li>Item 1</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Listas alfabéticas em maiúsculo.

Para criar listas alfabeticas maiúsculas ultizamos o TYPE A. 
<ol type = "A" start = "1">
    <li>Item 1</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Listas alfabeticas em minúsculo.

Para criar listas alfabeticas minúsculas ultizamos o TYPE a.

        
<ol type = "a" start = "2">
    <li>Item 1</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Lista com algarismos romanos em maiúsculo.

Para criar listas em algarismos romanos em maiúsculas ultizamos o TYPE I.

        
<ol type = "I" start = "3">
    <li>Item 1</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Listas com algarismos romanos minúsculo.

Para criar listas em algarismos romanos minúsculas ultizamos o TYPE i.

        
<ol type = "i" start = "4">
    <li>Item 1</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Listas não Ordenadas

Para criarmos uma unordered list, usamos a tag <ul> para delimitar a lista e a tag <li> para criar cada um de seus itens internos.

            
<ul>
    <li>Item 1</li>
</ul>

O marcador padrão é uma bolinha preta totalmente preenchida (circle), mas existe a opção de configurar a propiedade type da tag <ul> com os seguintes valores:

  • disc - padrão.
  • circle - Uma bola borda preta e sem preenchimento.
  • square - Um pequeno quadrado totalmente pintado.

Ficando:

        
<ul type = "square">
    <li>Item 1</li>
</ul>

Listas Mistas

Podemos também criar listas mistas, configurando listas dentro de outras listas.

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5

Também podemos juntar listas ordenadas com não ordenadas.

  1. Item 1
  2. Item 2
    • Item 2.1
    • Item 2.2
  3. Item 3
